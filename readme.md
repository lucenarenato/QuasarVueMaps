# Laravel + Vue 


## Folders
- Back end(Laravel)
- Front end(Vue.js)

## Como iniciar o projeto

```sh
chmod +x start.sh
./start.sh
```

## Como parar o projeto

```sh
chmod +x stop.sh
./stop.sh
```
